# Quantum_QKD_failure_inj

1- Com o SeQUeNCe já instalado, baixe o arquivo Json em um diretório qualquer e copie o path;
2- Baixe o os arquivos .py no mesmo diretório do SeQUeNCe e troque o pathe em run_qkd.py para onde você salvou o json;
3- Rode python run_qkd.py no diretório do SeQUeNCe;
4- Rode python -m pip install networkx matplotlib;
5- Rode python run_dataset_one_link.py para simular;
